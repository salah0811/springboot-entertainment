## 🛠️ 技术架构预览

前端：Vue 3.x + Element Plus + Axios + ECharts + WebRTC
后端：Spring Boot 2.7.x + MyBatis-Plus + Spring Security + JWT
数据库：MySQL 8.0 + Redis 6.x（缓存+Session）
实时通信：WebSocket + Netty + 声网Agora（音视频）

推荐算法：协同过滤 + 深度学习（内容推荐）
